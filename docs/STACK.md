# Tech Stack / Stack Tecnológico

## ES — Tecnologías principales
- **Backend/API:** Node.js + Express
- **Realtime:** WebSocket
- **Mensajería:** MQTT (Mosquitto)
- **Pipeline:** Telegraf
- **Time Series DB:** InfluxDB
- **Relacional:** MySQL
- **Dashboards:** Grafana
- **Web:** Next.js + TypeScript + Tailwind
- **App:** Flutter (Dart)
- **Infra:** Docker / Docker Compose (interno)
